File Checker 📂
Простая прога на Python для поиска дубликатов и проверки бэкапов. Всё хранится в SQLite, чтобы не хешировать одно и то же по сто раз.
Что умеет:
Сканирует папку и запоминает файлы.
Ищет копии по MD5-хешу.
Сравнивает оригинал с бэкапом (проверяет, всё ли на месте).
Как запустить:
bash

python main.py

Никаких лишних библиотек ставить не надо, только стандартный Python.
